# Contemplation Machine Release 1.9.3

**Release Date**: January 2, 2026

## Summary

Release 1.9.3 adds comprehensive setup and migration documentation for deploying the Contemplation Machine on the Asamadhi server, enabling smooth transitions from existing installations.

## Key Areas of Focus

### 1. Setup & Migration Documentation

- **Asamadhi Setup Guide**: Complete live-scripting instructions for initial deployment on Asamadhi
- **Data Migration**: Step-by-step instructions for migrating production data from old installations
- **Environment Configuration**: Automated copying of .env configuration from source installation
- **Verification Steps**: API health checks and data validation

## Features

- Initial setup with Docker deployment on Asamadhi
- Direct data migration from existing installations at `/Users/asamadhi/LocalProjects/ai-bootcamp/meditation-app`
- Preservation of API keys, instructions, presets, and audio cache
- Production-ready migration workflow

## Documentation Files

- `setup.org`: Complete setup and migration guide with executable live-scripting blocks
